Making a Contribution

Mahout is always looking for contributions, especially in the areas of documentation. See our How to contribute page for details.

Source Code

The source files are stored using Git, our page on version control has details on how to access the sourcecode.

Documentation

Javadoc and Scaladoc documentation is available online by module:

  • Mahout Math
  • Mahout Math Scala bindings
  • Mahout Spark bindings
  • Mahout Spark bindings shell
  • Mahout H2O backend Scaladoc
  • Mahout H2O backend Javadoc
  • Mahout HDFS
  • Mahout Map-Reduce
  • Mahout Examples
  • Mahout Integration

Issues

All bugs, improvements, pull requests, etc. should be logged in our issue tracker.

Continuous Integration

Mahout is continuously built on an hourly basis on the Apache Jenkins build system.

Board reports

Every three months Mahout submits a report to the Apache board. All of the drafts that get sent are checked into svn. See here:

https://svn.apache.org/repos/asf/mahout/pmc/board-reports/
